# Lista de Exercícios Docker Turma de abril (2025)

Este repositório contém uma lista de exercícios práticos para aprender e reforçar os conhecimentos em Docker, a plataforma de containers mais utilizada no mercado. Os exercícios são voltados para iniciantes e intermediários, com foco em comandos básicos, criação de imagens e manipulação de containers.
